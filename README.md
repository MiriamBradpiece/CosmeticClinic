# Cosmetic Clinic - in Progress*

## Overview
**Cosmetic Clinic** is a digital appointment management system for beauty salons, allowing clients to register, book appointments, and manage treatment packages. Clients with an active package can use it for booking, while those without must select a package before proceeding.

## Features
- User registration
- Appointment booking for various treatments
- Treatment package management
- Secure user authentication
- Modern and user-friendly interface

## Technologies Used
- **Backend**: C#  
- **Database**: SQL  
- **Frontend**: React  
- **State Management**: Redux
